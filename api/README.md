## Usage:

/start - Greeting

/currency - Currency Exchange rate. Example: 
> /currency UAH USD 10

/question - Ask any question. Based on Open AI (ChatGPT). Example: 
> /question Top movie titles 2023

/image - Image description. Based on Open AI (ChatGPT). Example: 
> /image orange sky
